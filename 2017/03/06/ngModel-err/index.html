Angular2中ngModel使用报错

发表于 2017-03-06 更新于 2019-11-27 分类于 Angular

问题：在angular2中，使用ngModel进行双向绑定的时候，老是会报错。
原因：经常忘记引入FormsModule
解决：引入FormsModule就可以了；angular2中如果想要使用相关的功能，一定记得引入对应的Module
