【转】Ubuntu下搜狗拼音异常解决

发表于 2017-03-27 更新于 2019-11-27 分类于 ubuntu

原文： ubuntu下搜狗输入法的输入框只显示英文不显示中文的问题

问题

Ubuntu下，搜狗拼音输入法能启动（系统托盘处有图标），但是打不出汉字，打字时选框不正常。

解决方法

删除搜狗拼音输入法的配置文件，并重启输入法。
执行如下指令

将两次搜索到的配置文件删除即可。
如果是单独文件，使用rm [文件名]指令，如果是目录，使用rm -R [目录名]指令。
要删除的文件夹有三个
SogouPY
SogouPY.users
sogou-qimpanel
全部删除之后就OK了
