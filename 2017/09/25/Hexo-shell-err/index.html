小错误导致脚本不执行

发表于 2017-09-25 更新于 2021-05-24 分类于 个人成长 ， Hexo

错误

自动部署hexo的脚本不执行了
经检查是脚本出错

原因

在windows上编辑之后就提交到git了，然后服务器上更新之后就不行了。
是因为windows末尾添加的乱七八糟符号。

解决

设置git以linux的方式提交。或，直接在linux下编写。只要没有windows添加的换行符之类的就好。
