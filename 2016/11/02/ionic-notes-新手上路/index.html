Ionic学习笔记--新手上路

发表于 2016-11-02 更新于 2019-11-27 分类于 ionic2
上一节已经完成了Ionic及其依赖的安装，现在，我们将一起学习 创建新应用、增加页面、在不同页面间导航 等等

原文链接

  1. 使用start命令初始化一个新的Ionic应用，通过添加--v2选项来使用Ionic2。我们还指定使用tutorial模板。Ionic2应用使用Typescript开发。
    $ ionic start MyIonic2Project tutorial --v2
    PS: 我们通过在命令中添加tutorial，来指定新生成的应用使用tutorial模板；如果不指定使用哪个模板，则Ionic2默认使用tabs模板。

  2. 启动，查看效果
