软件易用性的思考

发表于 2019-08-15 分类于 《设计心理学》 ， 读书笔记

引子

在一个付费的在线教育交流群里，有人已经付费了，面对APP却不知道自己会员身份的最基本权利。结合自己使用过程中的各种不便利，不禁感慨软件设计之艰难。

思考

在设计软件的时候，别指望用户会去读大段的使用说明。

做到识字知意，符合用户使用直觉，才是难得，这方面在《设计心理学》里有详细表述。

用什么态度对待用户呢？《华杉讲品牌营销》里有个理念，不要对用户做任何的预设和假设，即使假设，也是假设用户对你和你的产品、知识一无所知。如果能做到随便找个正常人，都能让对方一看到就明白，两三分钟就会用，那就算把设计做好了。

这方面，我觉得《滴答清单》做的就不错。一眼看到就能大体上知道怎么用，基本功能上很易上手。进阶功能呢？可以自己摸索，但是也提供了文档。文档写的怎么样呢？不说废话，不邀功，不自夸，简短篇幅教会你怎么用。这种文档不就是好文档吗？

Ubuntu Linux下修改HOST

发表于 2018-05-07 更新于 2019-11-27 分类于 ubuntu

原文：Ubuntu Linux下修改HOST

首先打开HOST文件

sudo vim /etc/hosts
添加你需要的域名

216.239.37.99 www.google.com
注意不要加上http:// 这样的协议前缀和/后缀等；

编辑后，你需要重新启动一下你的网络。

sudo /etc/init.d/networking restart
